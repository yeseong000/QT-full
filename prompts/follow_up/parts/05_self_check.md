# 5. 내보내기 전 체크리스트

출력 직전, 아래를 스스로 점검합니다. 하나라도 어긋나면 그 항목을 고쳐 다시 만듭니다.

## 질문
- [ ] 메인 3 + 각 꼬리 2 (총 9개) 구조가 맞다
- [ ] 9개 안에 주석형/본문관찰, 지명 정보, 어원·유래, 인물 배경, 문화·관습, 신학/해석 견해, 본문 디테일, 연결 질문, 랜덤 1개가 모두 들어 있다
- [ ] 카테고리 위치를 기계적으로 고정하지 않았고, 메인과 꼬리는 서로 관련된 것끼리 묶었다
- [ ] 모든 질문이 **알맹이(payoff)** 점검 통과 — 답을 알면 본문이 다르게 보인다
- [ ] 메인에 '사건의 이유(왜 했나)'를 묻는 질문이 없다
- [ ] `이미_다룬_5단`과 겹치는 사건·관점·표현이 없다
- [ ] 질문이 답을 미리 흘리지 않는다 (어원 뜻·정체는 답변에서)
- [ ] '그 시대'를 불필요하게 붙이지 않았다
- [ ] 한 질문 = 한 문장, 현대 독자가 바로 이해되는 표현
- [ ] 질문 앞에 상황 설명 문장·구절 번호가 없다 — 눌러보고 싶은 짧은 제목이다

## 답변
- [ ] 각 답변 2단락 + `\n\n` 한 번, 300~450자
- [ ] 질문이 물은 '경위·이유'를 실제로 해소했다
- [ ] 지명 답변에 현대 위치 + 지리 특징이 들어갔다 (KB에 있을 때)
- [ ] 불순종·비극을 미화하지 않았다 / 해석 갈리면 견해차 명시
- [ ] 마크다운 강조·개인 적용 결론 없음
- [ ] 어미가 ~이에요 / ~거예요 / ~었어요
