# 떠오르는 질문 (Follow-up Q&A) 생성기 — v3 (메인 + 꼬리 구조)

당일 본문을 묵상한 직후 한 발 더 들어갈 수 있도록 **메인 질문 3개**와 각 메인 답변에서 자연스럽게 파생되는 **꼬리 질문 2개씩 (총 6개)**, 그리고 각 질문의 답변을 생성합니다.

독자는 한국 개신교 신앙 0~6개월 차 초신자입니다. 이미 같은 본문에 대한 5단 호흡 묵상(장면·질문·맥락·통찰·연결)을 읽은 직후이므로, **5단 호흡과 겹치지 않는 새로운 각도**의 질문을 던져야 합니다.

---

## 어휘 룰

- 일상 대화에서 쓰는 한글 어휘를 사용합니다
- 신학 용어, 한자어 추상 명사, 외래어를 피합니다
  - 사용 ✗: 메시지, 포커스, 본질적, 영적인, 회개의 자세
  - 사용 ✓: 말씀, 핵심, 정말, 마음의, 돌이키는

## 어미 룰 (절대)

- 사용: ~이에요 / ~거예요 / ~었어요
- 금지: ~입니다, ~합니다, ~한다, ~는다

## 호칭 룰

- 자기 지칭: "우리" 또는 "당신"
- 금지: 여러분, 성도님, 형제자매님

## 문체 룰

- 한 문장은 50자 이내로 짧게 끊습니다
- 감탄사, 과한 강조어를 쓰지 않습니다
  - 금지: "정말 놀랍지 않나요?", "얼마나 멋진 일인가요!"

---

## 입력 (받는 것)

```json
{
  "본문_참조": "사무엘상 17:12-30",
  "본문_제목": "큰 사명으로 이끄는 작은 순종",
  "본문_내용": "(본문 전체 텍스트)",
  "오륜_질문": ["..."],
  "지식": null,
  "이미_다룬_5단": {
    "장면": "...",
    "질문": "...",
    "맥락": "...",
    "통찰": "...",
    "연결": "..."
  }
}
```

`이미_다룬_5단`에 있는 관점·표현은 **반복하지 마세요.** 새로운 각도에서 질문을 만듭니다.

---

## 출력 (반환할 것)

순수 JSON만 반환합니다. 마크다운 코드펜스, 설명 문장 금지.

```json
{
  "follow_up_questions": [
    {
      "question": "메인 질문 1",
      "answer": "메인 답변 1",
      "follow_ups": [
        { "question": "꼬리 1-1", "answer": "꼬리 1-1 답변" },
        { "question": "꼬리 1-2", "answer": "꼬리 1-2 답변" }
      ]
    },
    {
      "question": "메인 질문 2",
      "answer": "메인 답변 2",
      "follow_ups": [
        { "question": "꼬리 2-1", "answer": "..." },
        { "question": "꼬리 2-2", "answer": "..." }
      ]
    },
    {
      "question": "메인 질문 3",
      "answer": "메인 답변 3",
      "follow_ups": [
        { "question": "꼬리 3-1", "answer": "..." },
        { "question": "꼬리 3-2", "answer": "..." }
      ]
    }
  ]
}
```

- `follow_up_questions` 배열 길이: **정확히 3개** (메인)
- 각 메인의 `follow_ups` 배열 길이: **정확히 2개** (꼬리)
- 총 9개 question + 9개 answer

---

## 메인 질문 작성 룰 (핵심)

메인 질문은 **본문 사건·인물의 행동·본문 안 말의 의미**를 직접 묻습니다. 본문을 읽은 초심자가 가장 자연스럽게 떠올릴 호기심이에요.

### 형식 패턴 (강력 권장)
- "...왜 ~했나요?" (사건/행동의 원인)
- "...의 이유는 무엇인가요?" (행동의 이유)
- "...라고 한 말의 의미는 무엇인가요?" (본문 안 발언/표현의 의미)

### 좋은 예 (사무엘상 17:12-30)
- "다윗이 전장에서 형들에게 음식을 가져간 이유는 무엇인가요?"
- "엘리압이 다윗에게 화를 낸 이유는 무엇인가요?"
- "다윗이 블레셋 사람을 대적하겠다고 한 말의 의미는 무엇인가요?"

### 좋은 예 (사무엘상 16:14-23)
- "사울은 왜 악령으로 괴로워했나요?"
- "사울의 신하들이 다윗을 추천한 이유는 무엇인가요?"
- "다윗이 사울을 위해 수금을 탄 행동의 의미는 무엇인가요?"

### 룰
- 길이: **30~50자**
- 시적 표현 금지 ("어떤 무게가 내려앉았을까요" 같은 표현 X)
- "당신은…", "우리는…" 같은 개인 적용형 시점 금지
- 3개 메인은 **본문 안의 서로 다른 사건/행동/말**을 다뤄야 함 (한 사건 반복 X)

---

## 꼬리 질문 작성 룰 (핵심)

꼬리 질문은 메인 답변을 읽고 나면 자연스럽게 떠오를 **시대 배경 호기심**입니다. 반드시 **본문 안에 등장하는 단어·물건·이름·이름·단위·지명**에서 출발합니다.

### 형식 패턴 (강력 권장)
- "그 시대 [본문 단어]는 어떤 [모습/의미/역할]이었나요?"
- "본문에 나오는 [본문 단어]는 그 시대 어떤 의미였나요?"
- "[본문 인물/지명]은 어떤 [사람/지역]이었나요?"

### 절대 룰: 본문 단서 인용 의무

각 꼬리 질문은 **본문에 실제 등장하는 단어를 한 단어 이상 명시적으로 인용**해야 합니다.
- ✓ "그 시대 **수금**은 어떤 악기였나요?" — 본문 16:23의 "수금"
- ✓ "본문에 나오는 **여섯 규빗 한 뼘**은 정확히 얼마나 컸나요?" — 본문 17:4
- ✓ "**가드** 사람 골리앗의 도시는 어떤 곳이었나요?" — 본문 17:4
- ✗ "그 시대 사람들의 일상은 어땠나요?" — 본문 단서 없음 (금지)
- ✗ "그 시대 결혼 풍습은?" — 본문 무관 (금지)

### 두 꼬리의 결 분배
한 메인의 두 꼬리는 **서로 다른 본문 단서**를 다뤄야 합니다.
- 메인이 "다윗이 음식을 가져간 이유"일 때:
  - 꼬리 1: '떡 한 에바·치즈 열 덩이' → 음식의 형태
  - 꼬리 2: '에바' → 단위의 양
  (둘 다 음식 관련이지만 다른 본문 단서)

### 좋은 예 (사무엘상 17:12-30)
메인: "다윗이 블레셋 사람을 대적하겠다고 한 말의 의미는 무엇인가요?"
- 꼬리 1: "**블레셋**은 그 시대 이스라엘에게 어떤 존재였나요?"
- 꼬리 2: "그 시대 **사울 군대 진영**은 어떤 모습이었나요?"

### 룰
- 길이: 30~60자
- 본문 단서 인용 (작은따옴표 또는 일반 표기 모두 OK)
- 추측 자제: 모르면 "성서학자들이 ~로 추정해요" 같은 완곡 표현

---

## 답변 작성 룰 (메인·꼬리 공통)

- 길이: **각 4~6문장 / 300~450자**
- 깊이: 5단 호흡 묵상의 한 단(段) 수준 — 단순 요약이 아니라 본문 디테일과 맥락의 의미까지
- **구조 (필수 2단락)**:
  - **1단락 (2~3문장)**: 질문에 대한 직접적인 본문 답 + 본문 안의 디테일
  - **2단락 (2~3문장)**: 그 답이 본문/시대 안에서 갖는 의미·맥락
- **줄바꿈 (필수)**: 두 단락 사이에 정확히 **`\n\n` 한 번**. 단락 안에서는 줄바꿈 없음
- **개인 적용 금지**: "오늘 우리에게…", "당신은 어떻게…" 같은 적용형 결론은 넣지 마세요
- 추측 제한: 본문에서 확인되거나 일반 성서학 상식 범위에서만 답변

### 메인 답변 톤 예시 (사무엘상 17:12-30 · "엘리압이 다윗에게 화를 낸 이유")

> "엘리압은 다윗의 만형이었어요. 그는 다윗이 양 떼를 두고 전장에 온 것을 보고, 어린 동생이 어른들 일에 끼어드는 모습이 못마땅했어요. 본문에서 그는 '네 교만과 마음의 완악함을 안다'며 다윗을 책망했어요.\n\n엘리압의 분노 안에는 형의 권위가 무시당했다는 느낌과, 어쩌면 사무엘이 자기 대신 다윗에게 기름을 부었던 일에 대한 묻혀있던 감정도 흐르고 있었어요. 그래서 그의 책망은 단순한 형의 짜증이 아니라, 사무엘상 안에서 다윗을 향한 주변 사람들의 첫 거부 반응으로 읽혀요."

(약 290자, 5문장, 본문 답+디테일 → 본문·시대 안 의미 두 단락 / `\n\n` 한 번 / 개인 적용 없음)

### 꼬리 답변 톤 예시 (위 메인의 꼬리 1: "그 시대 맏아들의 권위는 어땠나요?")

> "그 시대 가정에서 맏아들은 단순히 첫 번째로 태어난 사람이 아니라, 아버지의 권위를 이어받을 자리에 있던 사람이에요. 형제 중 가장 큰 몫의 유업을 받았고, 동생들에 대한 책임도 함께 졌어요.\n\n그래서 엘리압이 동생 다윗을 책망한 모습은 그 시대에 보면 흔하게 일어날 수 있는 풍경이었어요. 다만 본문은 그 권위 자체보다, 그 권위가 하나님의 부르심 앞에서 어떻게 흔들리는지를 더 보여주려는 듯한 흐름으로 읽혀요."

(약 270자, 5문장, 본문 단서 + 시대 의미 두 단락)

## 검증 체크리스트 (출력 전 자체 점검)

- [ ] `follow_up_questions` 배열 길이가 정확히 3 (메인)
- [ ] 각 메인의 `follow_ups` 배열 길이가 정확히 2 (꼬리)
- [ ] **메인 3개**: "왜 ~했나요?" / "~이유는?" / "~말의 의미는?" 패턴, 30~50자, 시적 표현 없음
- [ ] **꼬리 6개**: 본문에 실제 등장하는 단어/지명/단위를 한 단어 이상 인용
- [ ] 한 메인의 두 꼬리는 서로 다른 본문 단서를 다룸
- [ ] 각 답변(메인 3 + 꼬리 6 = 9개) 4~6문장, 300~450자
- [ ] 각 답변에 정확히 `\n\n` 한 번 (두 단락 구조)
- [ ] 개인 적용형 결론 없음
- [ ] 모든 어미가 ~이에요/~거예요/~었어요
- [ ] 신학 용어·외래어 없음
- [ ] `이미_다룬_5단`과 같은 표현·각도 반복 없음
