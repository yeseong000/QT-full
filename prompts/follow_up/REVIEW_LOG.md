## R1 (2026-06-30) — 모듈화 + 1·2차 피드백 반영
- **단조·겹침 문제**: 메인을 '왜 했나(사건 이유)'에서 '한 층 파고드는 질문'으로 재정의 → `03`
- **7원칙 정립**: payoff·중립개방·'그시대'금지·현대눈높이(질문) / 물음표 해소·지명 현대좌표·미화금지(답변) → `03`·`04`
- **'숨은 아이러니형' 제외** → `03`
- **구체 예시**(가드·일 년 사 개월·맨 뒤·키 컸던 왕·시글락·사울 병기 든 자·요나단)를 좋은/나쁜 예시로 → `03`·`04`
- **KB 적극 활용 + KB 없으면 본문 디테일 우회** → `02`·`03`
- 단일 `system.md` → `parts/01~05` + `build.py` 모듈화. 라이브 코드 연결은 승인 후.

## R2 (2026-06-30) — 1차 전/후 비교(22~30) 결과 반영
- **원어 앵커링**: 음역(타르데마)만 불쑥 쓰지 말고 익숙한 번역어를 먼저 → `03`·검증
  - ✗ "타르데마가 사용된 다른 구절은?" → ✓ "'깊은 잠'이라는 히브리 원어 '타르데마'가…"
- **원어 질문은 '의미' 대신 '원어로 어떻게 써져 있나'** → `03`·검증
