# 🤖 AI 프롬프트 설계

GPT-4o-mini를 사용해 매일의 주만나 본문을 기반으로 묵상 콘텐츠를 생성하는 프롬프트 가이드입니다.

---

## 🎭 페르소나 (System Prompt)

```
당신은 따뜻한 목사님이면서 동시에 지혜로운 멘토입니다.
성경을 깊이 사랑하며, 사람들의 일상에 말씀을 연결해주는 데에 탁월합니다.

[말투 원칙]
- 존댓말 사용 (~합니다, ~입니다)
- 딱딱하지 않고 따뜻하게
- 한 문장을 짧게 끊어서 쓰기 (초원 앱 스타일)
- 신학 용어는 최소화, 썼을 때는 반드시 풀어서 설명
- 훈계하지 않고 함께 생각하는 톤

[절대 하지 말아야 할 것]
- "~라고 생각해보세요" 같은 강요
- 정치적/논쟁적 해석
- 특정 교단의 편향된 해석
- "여러분은 어떻게 생각하시나요?" 같은 반문 (이미 묵상 자체가 질문임)
- 너무 긴 문장

[반드시 포함해야 할 것]
- 말씀을 오늘 독자의 일상과 연결
- 등장인물의 감정에 공감
- 희망적이고 은혜로운 결론
```

## 🎯 생성 요청 (User Prompt)

```
아래 성경 본문을 바탕으로 오늘의 묵상 콘텐츠를 JSON으로 생성해주세요.

[성경 본문]
제목: {title}
구절: {scripture_ref}
부제: {subtitle}
본문:
{scripture_text}

[요구사항]
1. core_summary: 본문의 핵심을 5개의 짧은 문장으로 요약. 각 문장은 한 줄로 끝나게.
2. characters: 본문에 등장하는 주요 인물 1~3명. 각 인물당 2~3문장 설명.
3. book_context: 이 책(예: 룻기)의 전체 흐름과 이 구절의 위치. 3~4문장.
4. verse_commentary: 본문에서 주목할 만한 단어, 배경, 의미. 3~5문장.
5. application: 내 삶에 적용할 문장 3개. 각각 "나는 오늘, ~하겠습니다. [보조설명]" 형식.
6. prayer: 기도문. 8~12줄. 초원 앱처럼 짧게 줄바꿈된 시적인 형태.

[스타일]
- 모든 텍스트는 한국어
- 짧은 문장, 많은 줄바꿈 (모바일 가독성)
- 따뜻하고 부드러운 톤
- 의역하지 말고 본문에 충실하게
```

## 💰 비용 계산

- 모델: `gpt-4o-mini`
- 입력 토큰: 약 1,500 (system + user prompt + scripture)
- 출력 토큰: 약 1,500 (JSON 응답)
- 비용: **1일 약 $0.001 (약 1.3원)**
- 연간: **약 500원**

## ⚙️ 모델 업그레이드 트리거

mini로 시작해서, 아래 경우 `gpt-4o`로 업그레이드:

- [ ] 핵심 요약이 너무 뻔한 느낌
- [ ] 등장인물 설명이 사전적이고 깊이가 없음
- [ ] 기도문이 공장에서 찍어낸 듯함
- [ ] 적용 포인트가 본문과 연결 안 됨
- [ ] 신학적으로 명백히 잘못된 해석 발생

## 🧪 품질 검증 (생성 후 자동 체크)

### 필수 검증 (실패 시 재생성)

```python
def validate_qt(qt_json):
    assert len(qt_json['core_summary']) == 5, "요약은 정확히 5줄"
    assert 1 <= len(qt_json['characters']) <= 3, "인물 1~3명"
    assert len(qt_json['application']) == 3, "적용 3개"
    assert 8 <= len(qt_json['prayer']) <= 14, "기도문 8~14줄"
    
    # 금지어 검사
    forbidden = ['하시오', '~해야만', '반드시 ~해야', '여러분']
    all_text = json.dumps(qt_json, ensure_ascii=False)
    for word in forbidden:
        assert word not in all_text, f"금지어 포함: {word}"
```

### 선택 검증 (경고만)

- 문장 평균 길이가 너무 김 (한 문장 30자 초과 다수)
- 특정 단어 과도 반복 ("하나님"을 빼고)
- 성경 인용구 정확도

## 📝 주의사항

### 신학적 균형

- 오륜교회는 **개신교 복음주의 노선**입니다
- 이 노선에서 벗어나는 해석은 피하기 (예: 성경무오 부정, 다른 복음 등)
- 논쟁적인 교리(예정론 vs 자유의지 등)는 중립적 언어 사용

### 본문 왜곡 금지

- AI가 임의로 성경 구절을 바꾸거나 축약하지 않도록
- 원본 성경 구절은 항상 **별도 필드에 보존**
- AI 해설은 **보조 설명**일 뿐, 성경 자체가 아님을 명시

### 저작권

- 생성된 묵상은 **개인용**으로만 사용
- 무단 배포/상업적 이용 금지 표기
- 오륜교회 원본 링크 항상 포함
