# 🚶 사용자 플로우 - 4단계 묵상 여정

**목표**: 아침 3~5분 안에 말씀을 읽고, 묵상하고, 적용하고, 기록을 남기는 가벼운 QT 경험

---

## 🗺️ 전체 플로우 맵

```
[첫 방문]
    ↓
  이름 입력 (최초 1회)
    ↓
[🏠 홈] ← 설정 / 아카이브 접근 가능
    ↓
[1️⃣ 말씀] ← 본문 + 밑줄 긋기
    ↓
[2️⃣ 묵상] ← AI 해설 + 말씀 토글
    ↓
[3️⃣ 적용 + 기도] ← 적용 문장 + 기도문 + 기도 체크
    ↓
[4️⃣ 감정 + 메모] ← 감정 칩 + 자유 메모
    ↓
[✨ 완료] ← 연속일 달성 + 오늘의 회고
    ↓
  다시 [🏠 홈]
```

## 0️⃣ 첫 방문 (온보딩)

### 화면 구성

```
┌─────────────────────────────────┐
│                                  │
│         🕊️ 주만나 AI             │
│                                  │
│   매일 아침, 가장 따뜻한          │
│   묵상 파트너가 되어드릴게요       │
│                                  │
│   ┌────────────────────────┐   │
│   │  이름이 어떻게 되세요?    │   │
│   │                          │   │
│   │  [                    ]  │   │
│   └────────────────────────┘   │
│                                  │
│        [  시작하기 →  ]          │
│                                  │
└─────────────────────────────────┘
```

### 동작

- 이름 입력 → `localStorage.setItem('user.name', name)`
- iOS Safari인 경우, 다음 화면에서 **PWA 설치 안내 모달** 표시
- 이름 없이 진행 불가 (가벼운 장벽)

```
│                                  │
│  ╭──────────────────────────╮  │
│  │ 🔥 14일 연속 묵상 중         │  │
│  │ 🏆 총 47회 완료              │  │
│  ╰──────────────────────────╯  │
│                                  │
│  ╭──────────────────────────╮  │
│  │  오늘의 말씀                  │  │
│  │                              │  │

│  │  룻기 1:15-22                │  │

│  │     [ 묵상 시작하기 → ]      │  │
│  ╰──────────────────────────╯  │
│                                  │
│  📖 지난 묵상 모아보기 →          │
│                                  │
└─────────────────────────────────┘
```

### 요소

| 요소 | 설명 |
|---|---|
| 인사말 | 시간대별 변화 (`좋은 아침이에요`, `오후가 되었네요`, `편안한 밤 되세요`) |
| 이름 | `localStorage.user.name` |
| 날짜 | `YYYY년 M월 D일 요일` 형태 |
| 연속일 | `streakDays` (자정 넘기면 리셋 위험 있으므로 grace period 1일) |
| 총 완료 | `totalCompleted` |
| 제목 | `회복으로 나아가라` (오륜교회 제공) |
| 구절 범위 | `룻기 1:15-22` |
| CTA 버튼 | "묵상 시작하기" → 1단계로 이동 |
| 아카이브 링크 | 지난 묵상 리스트 페이지로 |

### 동작 상세

- **이미 오늘 완료했다면**: CTA가 "오늘의 묵상 다시 보기"로 변경
- **오늘 QT 로딩 실패 시**: "잠시 후 다시 시도해주세요" 표시 + 어제 묵상 링크 제공

## 1️⃣ 말씀 화면

**목적**: 성경 본문을 깨끗하게, 오롯이 읽는다.

### 화면 구성

```
┌─────────────────────────────────┐
│ ✕                    가가   🎵   │ ← 닫기 / 폰트크기 / 음악
│                                  │
│  말씀을 깊이 새겨보세요             │
│                                  │
│  ╭──────────────────────────╮  │
│  │ 오늘의 말씀      개역개정 ▾  │  │
│  │ 룻기 1:15-22                │  │
│  │                              │  │
│  │ 15 나오미가 또 이르되         │  │
│  │   보라 네 동서는 그의 백성과   │  │
│  │   그의 신들에게로 돌아가나니   │  │
│  │   너도 너의 동서를 따라       │  │
│  │   돌아가라 하니               │  │
│  │                              │  │
│  │ 16 룻이 이르되 내게 어머니를  │  │
│  │   떠나며 어머니를 따르지 말고 │  │
│  │   돌아가라 강권하지 마옵소서  │  │
│  │   ...                        │  │
│  │                              │  │
│  │ (전체 8절)                   │  │
│  ╰──────────────────────────╯  │
│                                  │
│                         ✏️        │ ← 밑줄 긋기 FAB
│                                  │
│  [  이전  ]    [  다음 →  ]      │
└─────────────────────────────────┘
```

### 요소 & 인터랙션

| 요소 | 동작 |
|---|---|
| ✕ (닫기) | 홈으로 이동 (확인 모달 없음) |
| 가가 (폰트) | 3단계 크기 전환 (소/중/대) → localStorage 저장 |
| 🎵 (음악) | 배경 찬양 재생 (선택) → 향후 구현 |
| 개역개정 ▾ | 번역본 선택 (MVP는 개역개정만) |
| 밑줄 긋기 ✏️ | 탭 → 밑줄 모드 진입 |

### 밑줄 긋기 플로우

1. ✏️ 탭 → 하단에 "마음에 드는 구절을 드래그하세요" 안내
2. 사용자가 텍스트 드래그
3. 노란 형광펜 느낌으로 하이라이트
4. 자동으로 localStorage에 저장
5. 다시 ✏️ 탭 또는 상단 "완료" → 모드 종료

- 스크롤해도 sticky 상태로 상단 고정

### 메모 동작

- textarea, 자동 리사이즈
- 최대 500자
- 입력 중 자동 저장 (debounce 1초)

## 📱 PWA 설치 안내 (iOS 사용자)

첫 방문 후 홈 화면 진입 시 한 번만 표시:

```
┌─────────────────────────────────┐
│                                  │
│       📱 앱처럼 사용하기          │
│                                  │
│   홈 화면에 추가하면 매일 아침    │
│   더 편하게 묵상할 수 있어요      │
│                                  │
│   1. 하단 [공유 📤] 버튼 탭      │
│   2. "홈 화면에 추가" 선택        │
│                                  │
│        [  다음에 안내  ]         │
│        [  이해했어요  ]          │
│                                  │
└─────────────────────────────────┘
```

- `localStorage.setItem('pwa.installPromptShown', 'true')`로 중복 방지
- Android는 Chrome 자동 배너 사용

## 🔄 이탈 방지 전략

- **중간에 나가도 진행 상태 저장**: 다시 들어오면 마지막 단계에서 이어서
- **"저장됨" 시그널**: 메모 입력 시 우상단 "저장됨 ✓" 잠깐 표시
- **강제 스킵 없음**: 단계마다 건너뛰고 싶으면 건너뛸 수 있게 (강요X)
