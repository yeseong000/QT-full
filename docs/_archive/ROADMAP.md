# 🗺️ 개발 로드맵

단계별로 MVP부터 확장 기능까지, 우선순위에 따라 정리했습니다.

---

## 🎯 전체 마일스톤

```
Phase 0: 프로젝트 세팅         (1일)
   ↓
Phase 1: 크롤링 MVP             (2~3일)
   ↓
Phase 2: AI 묵상 생성            (2일)
   ↓
Phase 3: 프론트엔드 MVP         (5~7일)   ← 여기까지 동작하는 앱
   ↓
Phase 4: 자동화 + 배포           (2일)
   ↓
Phase 5: PWA + 고도화            (3일)
   ↓
Phase 6: 추가 기능 (선택적)      (지속)
```

**전체 MVP까지**: 약 2~3주 (하루 2~3시간 작업 기준)

---

## 📦 Phase 0: 프로젝트 세팅 (1일)

### 할 일

- [ ] GitHub 리포지토리 생성 (`jumanna-qt`)
- [ ] 기본 폴더 구조 세팅
  ```
  /jumanna-qt
  ├── /docs              # 이 md 문서들
  ├── /scripts           # Python 크롤러
  ├── /data              # JSON 데이터
  ├── /public            # 정적 파일 (이미지 등)
  ├── /src               # 프론트엔드 소스
  ├── .github/workflows  # GitHub Actions
  └── README.md
  ```
- [ ] `.gitignore` 설정 (`.env`, `__pycache__`, `node_modules` 등)
- [ ] `requirements.txt` (Python) 작성
- [ ] `package.json` (JS) 초기화
- [ ] OpenAI API 키 발급 및 `.env` 파일 세팅

### 결과물
- 빈 프로젝트 껍데기

## 🎨 Phase 3: 프론트엔드 MVP (5~7일)

### 3.1 기본 세팅 (1일)

- [ ] Vite 프로젝트 초기화 (`npm create vite@latest`)
- [ ] Tailwind CSS 설치 + 설정
- [ ] 폴더 구조:
  ```
  /src
  ├── /components      (카드, 버튼, 칩 등)
  ├── /pages           (홈, 말씀, 묵상, 적용, 메모)
  ├── /lib             (localStorage helpers, date utils)
  ├── /styles          (global.css, tailwind.css)
  ├── main.js
  └── index.html
  ```
- [ ] 공통 컴포넌트 스타일링 (`UI_GUIDE.md` 참고)
- [ ] 라우팅 (SPA 라우터 또는 간단한 해시 라우팅)

### 3.2 홈 화면 (1일)

- [ ] 이름 입력 온보딩
- [ ] 날짜 + 인사말 표시
- [ ] 연속일/총횟수 표시
- [ ] 오늘의 말씀 카드
- [ ] 아카이브 링크
- [ ] "이미 오늘 완료했어요" 상태 처리

### 3.3 1단계 말씀 화면 (1일)

- [ ] JSON 데이터 fetch (`/data/qt/YYYY-MM-DD.json`)
- [ ] 성경 본문 렌더링
- [ ] 폰트 크기 조절 (소/중/대)
- [ ] 밑줄 긋기 기능
  - [ ] long-press 감지
  - [ ] 드래그 선택
  - [ ] 하이라이트 표시
  - [ ] localStorage 저장

### 3.4 2단계 묵상 화면 (1일)

- [ ] AI JSON 데이터 fetch (`/data/ai/YYYY-MM-DD.json`)
- [ ] 핵심 요약 / 등장인물 / 책 맥락 / 구절 해설 카드
- [ ] 상단 말씀 토글 (접힘/펼침)
- [ ] 스크롤 시 sticky 처리

### 3.5 3단계 적용+기도 화면 (1일)

- [ ] 적용 문장 3개 렌더링
- [ ] 기도문 렌더링 (시적 줄바꿈 유지)
- [ ] "기도했어요" 버튼
- [ ] 눌렀을 때 애니메이션 + 저장

### 3.6 4단계 감정+메모 화면 (0.5일)

- [ ] 감정 칩 9개 렌더링 (다중 선택)
- [ ] 메모 textarea (자동 리사이즈, 자동 저장)
- [ ] 완료 시 localStorage 업데이트
- [ ] 연속일 갱신 로직

### 3.7 완료 화면 + 아카이브 (0.5일)

- [ ] 완료 축하 화면 (연속일 애니메이션)
- [ ] 오늘의 밑줄 회고
- [ ] 아카이브 페이지 (날짜별 리스트)
- [ ] 아카이브 상세 (과거 묵상 재열람)

### 체크포인트
- ✅ 로컬에서 전체 플로우가 끊김 없이 동작
- ✅ 4단계를 다 완료하면 연속일이 오름
- ✅ 새로고침해도 진행 상태가 유지됨

## 🚀 Phase 6: 추가 기능 (선택적 / 지속)

우선순위 순서로 정리:

### 🥇 High Priority

- [ ] **연속일 마일스톤 축하** (10일/30일/100일 특별 화면)
- [ ] **밑줄 모음 페이지** (내가 그었던 모든 밑줄을 한 눈에)
- [ ] **감정 통계** (이번 달 내가 가장 많이 느낀 감정)
- [ ] **월별 묵상 요약** (지난 한 달의 여정을 돌아보기)

### 🥈 Medium Priority

- [ ] **배경 찬양 재생** (선택 가능한 BGM 리스트)
- [ ] **번역본 선택** (개역개정 외 NIV, 표준새번역 등)
- [ ] **공유 기능** (오늘의 밑줄을 이미지로 공유)
- [ ] **검색** (과거 묵상에서 키워드 찾기)

### 🥉 Low Priority

- [ ] **푸시 알림** (iOS 16.4+에서만)
- [ ] **그룹 기능** (가족/소그룹과 함께 묵상)
- [ ] **메모 마크다운 지원**
- [ ] **AI 챗봇** (오늘 말씀에 대해 질문하기)
- [ ] **다국어 지원** (영어 인터페이스)

### 💡 실험적 아이디어

- [ ] **목소리로 듣기** (TTS로 성경 본문 낭독)
- [ ] **손글씨 메모** (Apple Pencil 지원)
- [ ] **백업/동기화** (Supabase 무료 플랜 활용)
