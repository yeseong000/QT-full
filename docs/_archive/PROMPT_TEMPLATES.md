## 1. 새 화면 만들기

**언제 쓰나**: 2단계 묵상, 3단계 적용+기도 같은 **새 HTML 페이지**를 처음부터 만들 때.

**첨부 파일**: `BRIEFING.md` + `docs/USER_FLOW.md` + `src/step-1-scripture.html`(참고 패턴) + `data/ai/2026-04-22.json`(데이터 예시)

### 📋 복붙 템플릿

```
[작업 요청: N단계 화면 신규 제작]

현재 상태:
- 홈 화면과 1단계 말씀 화면은 완성됨
- 이번에 [N단계: 단계명] 화면을 새로 만들어야 함

참고 파일:
- 설계 명세: docs/USER_FLOW.md의 "[N단계 섹션]" 부분
- 패턴 기준: src/step-1-scripture.html (구조/스타일 그대로 따라할 것)
- 데이터: data/ai/2026-04-22.json 의 [사용할 필드: core_summary, characters 등]

만들 파일: src/step-[N]-[이름].html

요구사항:
1. design-tokens.css 변수만 사용 (임의 색상 금지)
2. 상단 헤더: [필요 요소 나열]
3. 본문 카드: [카드 종류 나열]
4. 하단 네비: 이전 → step-[N-1], 다음 → step-[N+1]
5. iPhone 14/15 Pro 최적화 (최대 폭 430px)

답변 방식:
- 먼저 구조를 표로 설명 (어떤 요소를 어디에 배치할지)
- 이후 HTML 파일 전체를 한 번에 제공
- localStorage 저장이 필요한 부분은 storage.js 확장안도 함께 제시
```

```

원하는 톤:
- 정중한 です/ます 체 (일본어의 경우)
```

### C. 이전 대화 이어갈 때 (새 세션)

```

```

## 📌 마무리 팁

3. **금지사항 위반은 초기에 잡기** — 한 번 허용하면 AI가 계속 그 방향으로 감.
4. **복잡한 작업은 쪼개기** — "전체 리팩터"보다 "파일 A 먼저" 순으로.
