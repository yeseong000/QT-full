# 🕊️ 주만나 AI 큐티 (Jumanna AI QT)

> 오륜교회 주만나 본문을 기반으로, AI가 매일의 묵상을 가이드해주는 모바일 웹 앱입니다.  
> **"바쁜 아침, 나만을 위한 가장 따뜻한 묵상 파트너"**

---

## 📌 프로젝트 한 줄 요약

매일 새벽, 로봇이 오륜교회 주만나 본문을 자동으로 가져오고 → GPT가 개인화된 묵상 가이드를 생성하고 → 초원 앱 스타일의 모바일 웹에서 4단계로 가볍게 QT를 할 수 있다.

---

## 🎯 핵심 철학

| 가치 | 설명 |
|---|---|
| **가벼움** | 아침 3~5분에 끝나는 묵상. 과하지 않게. |
| **분위기** | 초원 앱 같은 종이 질감, 따뜻한 크림 톤 |
| **기록** | 내가 밑줄 긋고 메모한 것들이 아카이브로 쌓여 성장감을 느낀다 |
| **자동화** | 사용자는 아무것도 하지 않아도 매일 최신 QT가 준비됨 |

---

## 🧱 4단계 플로우

```
🏠 홈 (날짜 + 오늘의 말씀 카드 + 연속일)
    ↓
1️⃣ 말씀  (성경 본문만 깔끔하게 - 밑줄 긋기 가능)
    ↓
2️⃣ 묵상  (AI 해설: 요약 + 등장인물 + 책 맥락 + 구절 해설)
    ↓       상단에 말씀 재확인 토글
3️⃣ 적용 + 기도  (실천 문장 + 기도문 + "기도했어요")
    ↓
4️⃣ 감정 + 메모  (감정 칩 선택 + 자유 메모)
    ↓
✨ 완료 (연속일 달성 + 내 밑줄 회고)
```

## 🏗️ 기술 스택

| 구분 | 선택 | 이유 |
|---|---|---|
| **프론트엔드** | Vanilla JS + Tailwind CSS | 가볍고, 디자인 시간 확보 |
| **AI** | OpenAI GPT-4o-mini | 저렴 + 충분한 품질 |
| **크롤링** | Python + BeautifulSoup | 오륜교회 주만나 페이지 |
| **자동화** | GitHub Actions (매일 새벽 5시) | 무료, 안정적 |
| **배포** | Vercel | 쉽고, PWA 캐싱 잘됨 |
| **저장소** | localStorage | 개인 메모/밑줄/연속일 |
| **앱화** | PWA (Manifest + Service Worker) | iOS/Android 홈 화면 설치 |
| **인증** | 이름만 입력 (localStorage) | 가볍게 |

## 🚀 빠른 시작

```bash
# 1. 로컬 개발
npm install
npm run dev

# 2. 오늘의 QT 수동 생성 (테스트용)
python scripts/fetch_qt.py
python scripts/generate_ai.py

# 3. 배포 (Vercel 자동)
git push origin main
```
